a

Tag usada para criar hyperlink e vincular uma página HTML com outra.
O atributo mais usado é o href.
Por padrão o conteúdo da tag é mostrado sublinhado.

Atributos

Estilo CSS padrão

a:link{
color: blue;
text-decoration: underline;
cursor: pointer;
}
a:visited {
color: #551a8b;
}
a:link:active{
color: red;
}
